# T6 · YouTube API key in Settings (SecretsService-backed, env fallback)

**Do:**
- `app/Config/YouTubeConfig.php` + `youtube.config.php`: read order = `SecretsService`
  (`youtube_data_api_key`) → `YOUTUBE_DATA_API_KEY` env fallback.
- Add a provider-credentials endpoint + a Settings section (`settings.view.php` + `settingsComponent`)
  following the **media-server create-form pattern**. Save via `SecretsService::put(...)`.
- **Never echo the key** — show set/unset state + a "replace" affordance only.

**Done when:** setting the key in the UI makes T5 available with no env change / restart; the key
never appears in responses or logs.
