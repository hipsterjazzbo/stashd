# Session Handoff

Create a concise handoff that another coding agent can use without reading the full transcript.

## Include

- Task goal.
- Current repo state summary.
- Files changed or inspected.
- Commands run and results.
- Decisions made.
- Known risks.
- Next recommended action.

## Avoid

- Full diffs.
- Full logs.
- Vague “continue from here” instructions.
- Raw tokens/secrets/URLs containing tokens.
