# Stashd Hermes entrypoint

Read `AGENTS.md` first.

For longer local-model setup notes, read:

```text
docs/ai/Hermes-Agent-Ollama-Setup.md
docs/ai/Stashd-Hermes-Agent-Reference.md
```

Hermes is a focused local coding assistant, not a frontier autonomous engineer.

Good Hermes tasks:

- explain unfamiliar code
- review selected files
- write narrow unit tests
- improve names/docblocks in 1-4 files
- draft plans for broader work
- summarize current architecture around a feature

Avoid broad implementation unless Hazel has narrowed the files and acceptance criteria.

Hardware reality: if running through Ollama on a 3060 12 GB box, inspect targeted files and avoid loading the whole repo.
